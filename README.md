# AI Squads Student Portal Redesign

A complete visual and functional redesign of the [AI Squads student portal](https://portal.aisquads.org), focusing on user experience, gamification, and a modern aesthetic.

## 🚀 Overview
The AI Squads portal is where students manage their bounties, track cohort progress, and build their profiles. This redesign aims to make the platform more engaging, intuitive, and professional.

## ✨ Key Features & Improvements
- **Gamified Experience**: Introduced an "Achievements" and "Badges" system to motivate students.
- **Improved Navigation**: A persistent sidebar for quick access to all vital sections.
- **Modern Dashboard**: A clean, data-driven overview of active bounties, XP, and rank.
- **Squad Connectivity**: Dedicated pages for cohort management and squad collaboration.
- **Responsive Design**: Built with Tailwind CSS to ensure a seamless experience across devices.

## 🛠 Technologies Used
- **HTML5**: Semantic structure for better accessibility.
- **Tailwind CSS**: For a modern, utility-first design system.
- **Font Awesome**: High-quality vector icons for better visual cues.
- **Google Fonts**: Custom typography for improved readability.

## 📁 Project Structure
- `index.html`: The central Dashboard.
- `bounties.html`: Browse and manage coding challenges.
- `cohort.html`: View squad members and cohort progress.
- `badges.html`: Gamification center showcasing student achievements.
- `profile.html`: Personal student profile with stats and skills.

## 🎨 Design Philosophy
The redesign uses a "Clean & Focused" philosophy. By using a white-space-heavy layout with indigo accents, we ensure that the content remains the primary focus while maintaining a high-energy "AI & Tech" feel.
